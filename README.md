# RL Data Dumper - Ferramenta para Extração de Dados do Rocket League

## Descrição

O RL Data Dumper é um projeto em C++ desenvolvido para o jogo Rocket League, com o objetivo de extrair dados do jogo em tempo de execução. O projeto utiliza técnicas e ferramentas de engenharia reversa e hooking de funções para obter os offsets das funções necessárias, permitindo a geração de um kit de desenvolvimento de software (SDK) para o jogo. Com esse SDK, o RL Data Dumper pode acessar e coletar informações sobre os jogadores em uma partida, bem como dados relacionados aos times.

Foi utilizado técnicas de engenharia reversa para analisar o código do jogo e identificar funções essenciais para a extração de dados.  
Foi utilizado hooking para interferir em funções específicas para interceptar e coletar dados em tempo real durante o jogo.  
Foi construido um kit de desenvolvimento de software (SDK) que oferece uma interface amigável para acessar e utilizar os dados coletados.  
Foram extraídos neste projeto informações dos Jogadores da partida e dados dos times.
